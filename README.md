# embedded-system-webapp
Embedded system assignment 3.

## 实现工具
* HTML5+js+css
    * <del>jQuery（若移动端支持），废案</del>
* apk & ipa打包
* 高德地图API
* 移动设备可访问硬件
    * GPS
    * 蓝牙

## 内容
* 简单的HTML布局 done
* GPS定位 done
* 现实交互型小游戏
    * 物品收集类RPG
    * 简单的用户系统
    * 等级
    * 生命
    * 角色属性值
* 简单的互动系统
    * 访问蓝牙，多用户交互
* 简单的战斗系统
* 简单的道具系统

## 测试
* web：Chromium 63.0.3239.132
* Android：HUAWEI P10，Android 6.0
* iOS：iPhone 6，iOS 11.4.1
